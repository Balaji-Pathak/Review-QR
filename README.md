# Review-QR

A simple landing page project for a Review QR experience with mock product data.

## Structure

- index.html — landing page markup
- style.css — page styling
- script.js — renders shop cards from mock data
- data/shop.js — temporary product data
- assets/logo.png — placeholder logo

## Run locally

Open index.html in a browser, or serve the folder with a static server such as:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000
